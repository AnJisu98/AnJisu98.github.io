### 한계점       
  유튜브 실시간 스트리밍 댓글로 데이터의 대상이 정확했으므로 데이터 크롤링, 라벨링에 한계가 있었다     
  데이터의 양, 질에 영향을 많이 받는 딥러닝(자연어처리) 특성상 좋은 성능을 보이진 않았음
